## 简介

SJMC SMP2 Satellite is a SJMCL extension that displays the real-time satellite map of the SJMC SMP2 server directly in the launcher.

⚠️ This extension is for the SJMC SMP2 server only.

Two versions available:
- **English** (org.yoshino.sjmc_smp2_satellite)
- **中文版** (org.yoshino.sjmc_smp2_satellite.zh)

## Features

- 🗺️ Real-time SMP2 server satellite map
- 🔄 Multi-dimension switching (Overworld, Nether, End)
- 🎮 2D / 3D view switching
- 📍 Player location markers

## Installation

1. Download the .sjmclx file from Releases
2. Open SJMCL → Settings → Extension Management → Import
3. View from the home card or extension page

## Compatibility

- Minimum launcher version: 1.1.3
